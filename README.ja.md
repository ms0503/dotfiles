# ms0503's dotfiles

- [English](README.md)

Nix flakesで管理している環境設定。

# 使い方

## 必要なもの

- Nix (下記のexperimental-featuresが必要)
  - `nix-command`
  - `flakes`
- `nh`

## コマンド

```shell
$ # OSの同期(NixOSのみ)
$ nh os switch .
$ # ホームの同期
$ nh home switch .
```
